# Image Registry Map

This document maps image keys to their usage across the website.

## Logo Variants

- **logo.primary**: Main logo used in header, footer, and hero sections
- **logo.white**: White variant for dark backgrounds (hero overlays, dark footer)
- **logo.grey**: Grey variant for watermarks and light sections
- **logo.favicon**: Site favicon and pinned tabs

## Home Page Images

- **home.heroBg**: Hero section background image
- **home.features1**: "Full Accounting Service" feature block
- **home.features2**: "Small Business Specialists" feature block

## Service Page Images

- **services.compliance**: Compliance Services page
- **services.support**: Support Services page
- **services.consultancy**: Consultancy & Systems Advice page
- **services.startup**: Business Start-Up page
- **services.taxation**: Taxation page
- **services.corporationTax**: Corporation Tax Services page
- **services.personalTax**: Personal Tax page
- **services.vat**: VAT page
- **services.niPaye**: NI and PAYE page
- **services.hmrc**: HMRC Investigations page

## Updating Images

To update any image:
1. Edit `/assets/images/registry.json`
2. Update the `url` field for the desired image key
3. Optionally update the `alt` text for better accessibility
4. Save the file - changes will be reflected immediately
