# Image Registry Map

This document maps image keys to their usage across the website. All images are sourced from `registry.json`.

## Logo Variants

- **logo.primary**: Main logo for light backgrounds (Header, SEO).
- **logo.white**: White variant for dark backgrounds (Header on Hero, Footer).
- **logo.grey**: Grey variant, intended for watermarks.
- **logo.favicon**: Site favicon.

## Home Page Images

- **home.heroBg**: Background for the main hero section on the Home page.
- **home.features1**: Image for the "Full Accounting Service" feature block on the Home page.
- **home.features2**: Image for the "Small Business Specialists" feature block on the Home page.

## About Page Images

- **about.team**: Main image used on the About Us page.

## Service Page Images

Used in `ServiceCard` components on the Home page and the main Services page. Also used as primary images on individual service detail pages.

- **services.compliance**: Compliance Services
- **services.support**: Support Services
- **services.consultancy**: Consultancy & Systems Advice
- **services.startup**: Business Start-Up
- **services.taxation**: Taxation
- **services.corporationTax**: Corporation Tax Services
- **services.personalTax**: Personal Tax
- **services.vat**: VAT
- **services.niPaye**: NI and PAYE
- **services.hmrc**: HMRC Investigations
